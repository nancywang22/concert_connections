# concert connections

Concert Connections is a web application that lets music fans log concerts theyâ€™ve attended, share photos and captions from their experiences, and explore posts from other users. 

Made with Node.js + Express + MongoDB Atlas + React

Check it out at: https://concert-connections.vercel.app/

# Features

User Authentication: Register and log in using email/password with JWT-based authentication.

Concert Logging: Users can search for an artist, select a concert (with city, date, and venue), and upload a post with an optional image and caption.

Post Feed: Main page displays posts from other users, showing concert details, images, captions, and usernames.
